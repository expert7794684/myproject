<h2>功能</h2>
<ul>
<li>现代易用的界面</li>
</ul>
<h3>闹钟</h3>
<ul>
<li>可自定义的时间计划（每天，每周，特定星期与日期，日期范围）</li>
<li>设置铃声、渐响音量与振动</li>
<li>配置贪睡时长与最大次数</li>
<li>闹钟任务（数学题，打字，数列及更多）</li>
<li>筛选闹钟（所有，今天，明天，稍后再响，已禁用，已完成）</li>
</ul>
<h3>时钟</h3>
<ul>
<li>可自定义的时钟显示</li>
<li>显示相对时差的世界时钟</li>
<li>搜索并添加城市</li>
</ul>
<h3>计时器</h3>
<ul>
<li>设置铃声、渐响音量与振动</li>
<li>计时器预设</li>
<li>筛选计时器（所有，进行中，已暂停，已停止） </li>
</ul>
<h3>秒表</h3>
<ul>
<li>显示每圈用时与总用时</li>
<li>圈与圈比较</li>
</ul
<h3>外观</h3>
<ul>
<li>Material You 主题</li>
<li>高度可定制的颜色主题</li>
<li>高度可定制的样式主题</li>
</ul>
